SpecJava use of sub-classes

This form needs to be viewed using a Java enabled browser such as NetScape 2.0!

This demo demonstrates the use of custom components from SpecJava. In this case, any java "applet" may be embedded as a component (thanks to Cris Perdue). The new component is created by dragging a canvas into the desired location, then using the sub-class field on the option sheet to specify its class, and initialization method (if any).
