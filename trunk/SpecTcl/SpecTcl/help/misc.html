Miscellaneous Stuff

  • SpecTcl looks at the following environment variables:
    SPECTCL_DIR
    The directory that contains the SpecTcl source files
    SPECTCL_SIZE
    If specified, each font used by SpecTcl is increased (or decreased) in size by this amount (in points).
    SPECTCL_HELP
    The directory containing the SpecTcl help files
    SPECTCL_RC
    An alternate location for the SpecTcl startup file.
