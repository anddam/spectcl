SpecTcl Tutorial - part 2

As the soon-to-be-created widget is dragged over the canvas, three kinds of feedback is provided to the user:

  • The row and column indicators light up for the cell that the widget is about to be dropped into,
  • The cursor shape changes to indicate what will happen if the mouse button is released at the current position.
  • The Message area prints the current row and column number, and indicates if the current cell is occupied or not.
